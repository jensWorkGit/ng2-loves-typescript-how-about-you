async/await
  • Asynchrone Programmierung ist schwierig
    • Stark verschachtelter Code
    • Oft schlecht lesbar
  • Async/Await macht Code wieder flach!
    • ES Proposal
    • Leider nicht in ES2016, hoffentlich 2017
    • TypeScript unterstüzte schon für target: "es6"
    • Ab 2.1 auch für ES3 und ES5

  await doSomethingAsync();

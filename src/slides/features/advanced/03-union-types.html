Union Types
  • Union Types sind die Vereinigung von 2 Typen
    • Werden in der Typ Signatur über | verbunden
    • Zuweisbar: beide Typen
    • Aufrufbar: was in beiden Typen gleich definiert ist
let campusPerson: Student | Professor;

campusPerson = new Student("Max", 123456); // OK
campusPerson = new Professor("Hugo"); // OK
campusPerson = "Paul"; // Fehler
  • Wichtig für JavaScript-Welt
    • Kein Überladen von Methoden
    • Eine Methode nimmt A oder B entgegen (z.B. string | string[])
    • Innerhalb der Methode Typ-Prüfung (Array.isArray)

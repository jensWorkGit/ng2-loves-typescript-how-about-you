Klassen & Interfaces
  • Interfaces und Klassen können sinnvoll kombiniert werden
  • Interfaces definieren Schnittstellen und fördern die Wiederverwendbarkeit 
    interface Labelled {
    getLabel(): string;
}
  • Klassen und Objekte werden per structural typing mit Interfaces verglichen 
    let withLabel: Labelled = new Student("Max", 123456);
  • Klassen können Interfaces explizit implementieren 
    class Student implements Labelled {}
    • Schon bei Erstellung sichergestellt, dass Klasse strukturell zu Interface kompatibel

Typisierung
  • Wann wird der Typ geprüft?
    • statische Typisierung: String s = new String();
    • dynamische Typisierung: var s = "a"; s = 3;
  • Wir streng sind die Typen?
    • Wie viel wird automatisch konvertiert?
    • schwache Typisierung: s = "Zahl " + 5;
    • starke Typisierung: s = "Zahl " + str(5)
  • JavaScript: dynamisch, sehr schwach
  • TypeScript: statisch, etwas stärker aber immer noch schwach

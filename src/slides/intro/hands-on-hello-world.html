Hands-on: Hello World!
  1. Lege in der IDE ein neues Projekt an (Static Web bzw. Empty Project)
  2. Lege im Projekt eine Datei tsconfig.json an (Datei / Neu / tsconfig.json)
  3. Konfiguriere den TypeScript Support
    • Datei / Einstellungen / Sprachen & Frameworks / TypeScript
    • TypeScript Compiler einschalten
    • tsconfig.json verwenden
  4. Lege die Datei test.ts mit folgendem Inhalt an: 
    console.log("Hello World!");
  5. Führe die Datei test.js aus (Rechtsklick -> Ausführen)
    (js Datei in IDE-Explorer als Kind von ts)
